# 🌸 观测即意义 (Shiyun Observer)

一个基于 LLM 的交互式艺术体验，探索审美与意义构建。

## ✨ 特性

- **LLM 驱动**：使用 OpenAI GPT 或 Anthropic Claude 生成诗意文本
- **动态交互**：诗句实时生成、浮现、消失
- **流式洞察**：AI 实时分析你的审美偏好并生成深度洞察报告
- **游戏化体验**：一步步引导的沉浸式体验

## 🚀 快速开始

### 1. 安装依赖

\`\`\`bash
npm install
\`\`\`

### 2. 启动开发服务器

\`\`\`bash
npm run dev
\`\`\`

### 3. 打开浏览器

访问 \`http://localhost:5173\`

### 4. 开始体验

1. **选择 AI 模型**：OpenAI (GPT-4) 或 Anthropic (Claude)
2. **输入 API Key**：你的 API Key 仅用于当前会话，不会被存储
3. **（可选）自定义 Base URL**：
   - 用于代理服务器
   - 或兼容 OpenAI API 的第三方服务
   - 例如：`https://api.openai-proxy.com`
4. **（可选）自定义模型**：
   - OpenAI: 默认 `gpt-4o-mini`
   - Anthropic: 默认 `claude-3-haiku-20240307`
5. **开始观测**：
   - 诗句会从虚空中浮现
   - 点击你喜欢的诗句
   - 等待诗句消失
6. **结束观测**：点击"结束观测"按钮
7. **查看洞察**：AI 将流式输出对你的深度分析

## 🎮 游戏流程

\`\`\`
欢迎页面
  → 输入 API Key（可选择记住配置）
  → 开始观测
  → 移动鼠标探索诗云
  → 诗句从鼠标方向浮现
  → 点击喜欢的诗句（发光）
  → 诗句逐渐消失
  → 结束观测
  → AI 流式生成洞察报告
  → 重新开始（配置已保存）
\`\`\`

## 💡 交互提示

- **移动鼠标**：诗句会在鼠标方向生成
- **停止移动**：诗句会逐渐消失
- **点击诗句**：诗句会发光并影响后续生成
- **双击空白**：无操作（未来可扩展）
- **1.5秒冷却**：避免诗句生成过快

### 配置保存功能
- ✅ **自动保存**：勾选"记住配置"后，配置将保存在浏览器本地
- ✅ **自动填充**：下次打开自动填充上次使用的配置
- ✅ **安全性**：API Key 仅存储在你的浏览器中，不会上传到任何服务器
- ✅ **清除功能**：点击"清除"按钮可删除保存的配置

### 🔑 API Key 获取

### OpenAI
1. 访问 https://platform.openai.com/api-keys
2. 创建新的 API Key
3. 默认模型：gpt-4o-mini（可在界面中自定义）
4. 如需使用代理，填写 Base URL

### Anthropic
1. 访问 https://console.anthropic.com/
2. 创建新的 API Key
3. 默认模型：claude-3-haiku-20240307（可在界面中自定义）
4. 如需使用代理，填写 Base URL

### 第三方服务
支持任何兼容 OpenAI API 的第三方服务：
- 在"选择 AI 模型"中选择"OpenAI"
- 填写第三方服务的 API Key
- 填写第三方服务的 Base URL
- （可选）填写自定义模型名称

## 🎨 核心功能

### 诗句生成
- 每 2 秒生成一句新诗
- 诗句存在 15 秒（12 秒显示 + 3 秒淡出）
- 基于用户点击历史优化生成

### 交互反馈
- **悬停**：诗句显示在屏幕中央
- **点击**：诗句发光，影响后续生成
- **自动旋转**：3D 场景缓慢旋转

### AI 洞察
- 分析点击的诗句
- 计算探索性和收敛性
- 生成 200-300 字的深度洞察
- 流式输出，实时显示

## 🌟 设计理念

> "观测即意义" — 你的每一次点击，都在赋予文本以意义

这个项目探索了：
- **生成与消逝**：诗句如云烟般浮现又消散
- **选择与偏好**：点击即是赋义
- **学习与适应**：AI 根据你的选择调整生成
- **洞察与反思**：最终报告是对你审美的一次照镜
